LANlord – Terminal Network Scanner

A fast, keyboard-first Angry IP Scanner–style tool built with Python + Textual.

Features
- Auto-detect local subnet
- Alive + Dead host detection
- Port scanning
- Click / select IP for details
- Fully async
- Keyboard driven

Run
pip install -r requirements.txt
python app.py

Keys
s → scan
r → reset
q → quit
↑ ↓ → select IP
